# The Textile Bar — Studio Pro

Application web de gestion BAT, devis/factures, tarifs et stock.

## Déploiement sur Vercel

1. Uploadez ce dossier sur GitHub
2. Connectez le repo sur vercel.com
3. Vercel détecte automatiquement React → cliquez Deploy
4. Configurez votre domaine dans Settings > Domains

## Développement local

```bash
npm install
npm start
```

## Structure
- `src/App.jsx` — Application complète
- `public/` — Fichiers statiques
- `vercel.json` — Configuration routing
